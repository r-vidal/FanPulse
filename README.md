- Repeat play rate
- Playlist adds

- Days since last listen
- Consistency over time
- Seasonal patterns

**Component 3: Social Engagement (20% weight)**
- Instagram comments, shares, mentions
- TikTok UGC creation
- YouTube comments
- Story mentions

**Component 4: Monetization History (10% weight)**
- Concert ticket purchases
- Merchandise orders
- Streaming loyalty (premium vs free)

### 3. Momentum Index

**Purpose:** Detect growth trajectories before they're obvious. Think of it as a "stock momentum indicator" for artists.

**How It Works:**

The Momentum Index is a score from 0-10 that predicts an artist's trajectory:

**0-3:** Declining (action needed)  
**3-5:** Stagnant (need strategy shift)  
**5-7:** Steady growth (maintain course)  
**7-9:** Accelerating (capitalize on momentum)  
**9-10:** Viral/breaking (move fast)

**Calculation Components:**

```python
Momentum Index = f(
    Stream Growth Rate,        # 30% weight
    Engagement Velocity,       # 25% weight
    Playlist Momentum,         # 20% weight
    Social Virality,          # 15% weight
    Press & Discovery          # 10% weight
)
```

**Detailed Breakdown:**

**1. Stream Growth Rate (30%)**
```
Current 7-day MA vs 21-day MA
Acceleration coefficient
Trend strength
Anomaly detection (spikes vs organic growth)

Example:
Week 1-3 avg: 10k streams/day
Week 4 avg: 15k streams/day
Growth rate: +50%
Momentum contribution: +2.5 points
```

**2. Engagement Velocity (25%)**
```
Instagram: Likes, comments, shares per post
TikTok: Video creation rate using track
YouTube: Comment sentiment, watch time
Spotify: Saves, playlist adds, skip rate

Example:
Instagram engagement: 4.8% â†’ 6.2% (+29%)
TikTok UGC: 3 videos â†’ 12 videos
Momentum contribution: +2.1 points
```

**3. Playlist Momentum (20%)**
```
New playlist adds (algorithmic vs editorial)
Playlist rank improvements
Total playlist reach change
Playlist listener retention

Example:
Added to 45 new playlists this week
"New Music Friday France" (#87 â†’ #12)
Momentum contribution: +1.8 points
```

**4. Social Virality (15%)**
```
TikTok sound usage growth rate
Twitter/X mentions spike detection
Instagram Reels featuring track
Reddit community discussions

Example:
TikTok: Sound used in 850 videos (+400% week-over-week)
```

```python
def score_release_date(artist_id, date):
    score = 50  # Baseline
    
    # Factor 1: Artist Momentum (25%)
    momentum = get_predicted_momentum(artist_id, date)
    if momentum > 7:
        score += 25
    elif momentum > 5:
        score += 15
    else:
        score += 5
    
    # Factor 2: Fan Engagement (20%)
    fan_activity = get_fan_activity_pattern(artist_id, date)

    score += day_of_week_score * 20
    
    # Factor 3: Competition (20%)

    competition_penalty = len(competitors) * 2
    score -= min(20, competition_penalty)
    
    # Factor 4: Seasonal (15%)
    seasonal_multiplier = get_seasonal_factor(date, artist_id.genre)
    score += seasonal_multiplier * 15
    
    # Factor 5: Platform Algorithms (10%)
    algo_score = check_platform_algorithms(date)
    score += algo_score * 10
    
    # Factor 6: Special Dates (10%)
    if is_bad_date(date):  # Black Friday, Christmas Day, etc.

    if is_good_date(date):  # Genre-specific good dates
        score += 10
    
    return min(100, max(0, score))
```
